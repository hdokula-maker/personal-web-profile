Projects

1. Fire Fighting Robot

The Fire Fighting Robot is an embedded systems project designed to detect and extinguish fire automatically. The robot uses flame sensors to identify the presence and direction of fire. An Arduino Uno microcontroller processes the sensor data and controls the movement of the robot using DC motors and motor driver modules. Once the robot reaches the fire source, a water pump mechanism is activated to spray water and suppress the fire. The project helped in understanding robotics, sensor integration, automation, embedded systems, and microcontroller programming. The project was shortlisted among the Top 25 teams out of more than 100 participating teams for external evaluation.

Technologies Used: Arduino Uno, Flame Sensors, DC Motors, Motor Driver Module (L293D), Water Pump, Chassis, Battery Module, Embedded Systems, Robotics

2. Idea Companion - Smart Idea Management System

Idea Companion is a Java-based application developed to help users store, organize, manage, and evaluate ideas efficiently. The system allows users to add, edit, delete, search, categorize, and rate ideas based on priority and importance. Data is stored permanently using file handling techniques to ensure information is preserved across sessions. The application demonstrates Object-Oriented Programming concepts such as encapsulation, inheritance, and polymorphism while utilizing efficient data structures for idea management.

Technologies Used: Java, OOP, ArrayList, HashMap, File Handling, Serialization, Exception Handling

3. Student Data Management System

The Student Data Management System is a data structures-based application developed to manage student records efficiently. The system supports operations such as insertion, deletion, updating, searching, sorting, and grouping of student information. Linked Lists were used for dynamic record storage, Binary Search Trees (BST) enabled efficient searching and retrieval, and Hash Tables were used for faster access to student records. This project strengthened my understanding of Data Structures and Algorithms and their practical applications.

Technologies Used: C Programming, Linked Lists, Binary Search Trees (BST), Hash Tables, Data Structures and Algorithms

4. Mental Wellness and Well-Being Application

The Mental Wellness and Well-Being Application was developed as part of the CodeHer Hackathon to support emotional health and promote positive well-being. The application provides features such as journaling, mood tracking, breathing exercises, and interactive stress-relief activities. The platform was designed with a user-friendly interface that encourages users to develop healthy habits and manage stress effectively. The project focused on creating a positive and engaging digital experience for users.

Technologies Used: HTML, Frontend Development, User Interface Design
